# CineMatch 
CineMatch – Hybrid Movie Recommendation System

CineMatch is a desktop-based movie recommendation system built using C# and MySQL. It combines user preferences, rating history, and trending data to generate personalized movie suggestions.

🚀 Features
🔐 User authentication (Login/Register)
⭐ Rate movies and update ratings
🎯 Personalized recommendations (genre-based, top-rated, trending)
📊 User profile with rating history and preferences
🧠 Hybrid recommendation logic using SQL + user behavior
💻 Clean Windows Forms UI
🛠️ Tech Stack
Frontend: Windows Forms (.NET)
Backend: C# (ADO.NET)
Database: MySQL
Tools: MySQL Workbench, Visual Studio
🗄️ Database Setup
Open MySQL Workbench
Create a database named cinematch
Import the cinematch.sql file
Update database credentials in DB.cs
▶️ How to Run
Clone the repository
Open solution in Visual Studio
Build and run the project
Login using demo users or register a new account
